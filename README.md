# FirstOnlineGame
なんとなく文化祭やVEEにあるような、数人だけで遊べるオンラインゲームを作りたくなったのでここを作りました。Gravityみたいなの。

# ゲーム概要(プロト版)
プレイヤーは逃げる側(Fruit)と鬼(Fork)に分かれて鬼ごっこをする

ForkはFruitを捕まえたら勝ち、FruitはForkから逃げ切ったら勝ち

制限時間は10秒

# わからないこと
いまは特にない

# いま取り組んでいること
